для работы нужен файл SQL.txt с запросом, в запросе должна быть конструкция ID in ~ID~ например:
Select * from table where table_id in ~ID~

и файл SQLParameters.txt для чтения данных параметров подключения логина и пароля в формате:
host;port;sid;user;pwr

Программа запускается с 2 параметрами, 1 параметр - это количество потоков, 2 - это размер одной части
